Yatrika - Travel Guide Website

Yatrika is a modern and responsive Travel Guide Website built using HTML and CSS.  
The project is designed with a clean UI, responsive layouts, interactive hover effects, and organized folder structure.

This project showcases frontend development skills using Flexbox, CSS Grid, responsive design, animations, and modern UI styling.

---

# 🚀 Features

- Modern travel website homepage
- Responsive design for mobile, tablet, and desktop
- Sticky navigation bar
- Hero section with background image
- Search and filter section
- Featured destinations section
- Popular hotels section
- Top restaurants section
- Hover animations and transitions
- Organized image folder structure
- Flexbox and CSS Grid layouts
- Professional UI design
